# 치안/범죄 예방 API 검토 메모

기준일: 2026-03-09

검토 기준:
- Brave Search + Perplexity로 최신 메타데이터 확인
- 공식 `data.go.kr`, `safemap.go.kr`, `police.go.kr` 기준으로만 정리
- Data Layer에 바로 붙일 수 있는 `WMS`, `좌표형 point`, `공간정보` 보유 여부를 우선 확인

## 1. 현재 상태

- 현재 `crime` 도메인은 `범죄 위험 히트맵`, `순찰 집중선`이 mock 상태
- 공개 API 기준으로 `실시간 순찰 경로`, `실시간 출동 위치`, `실시간 범죄 발생 위치`는 확인하지 못함
- 따라서 1차 구현은 `생활안전지도 WMS + 치안시설/비상벨 위치정보` 중심이 맞음

## 2. 우선 구현 후보

- [ ] `범죄주의구간`
  - API: [행정안전부_생활안전지도 범죄주의구간(전체, WMS)](https://www.data.go.kr/data/15101797/openapi.do)
  - 적합성: 높음
  - 이유: 최근 1년 5대 범죄 밀도분석 결과를 도로상 10등급으로 표현하는 전국형 WMS
  - 권장 사용: 현재 `범죄 위험 히트맵` 대체용 메인 레이어
  - 비고: `heatmap`보다는 `WMS overlay` 성격이 강함

- [ ] `여성밤길 치안안전`
  - API: [행정안전부_생활안전지도 여성밤길치안안전(전체, WMS)](https://www.data.go.kr/data/15101813/openapi.do)
  - 적합성: 높음
  - 이유: 밤 시간대 여성 대상 범죄 밀도 분석 레이어
  - 권장 사용: 독립 레이어 또는 `범죄주의구간`의 특화 서브 메뉴
  - 비고: 여성 안전, 귀가 지원, 야간 보행 안전 시나리오에 적합

- [ ] `치안안전시설`
  - API: [행정안전부_생활안전지도 치안안전시설](https://www.data.go.kr/data/15101889/openapi.do)
  - 적합성: 높음
  - 이유: 전국 경찰서, 지구대, 파출소, 치안센터 정보를 제공
  - 권장 사용: `경찰관서 위치` 메뉴
  - 확인 포인트: 공식 설명상 관할경찰청, 시설분류, 시설명, 주소와 함께 `UTMK 위치정보` 제공
  - 비고: point 레이어로 쓰려면 WGS84 변환이 필요할 수 있음

- [ ] `안전비상벨`
  - API: [행정안전부_생활안전지도 안전비상벨 위치정보](https://www.data.go.kr/data/15149530/openapi.do)
  - 적합성: 높음
  - 이유: 범죄예방용 비상벨의 위치, 주소, 설치위치, 점검일자 등을 제공
  - 권장 사용: `비상벨 위치` 메뉴
  - 확인 포인트: 공식 설명상 위도, 경도 제공
  - 비고: 편의점/소방 목적 비상벨은 제외

- [ ] `범죄예방환경설계(CPTED)`
  - API: [행정안전부_생활안전지도 범죄예방환경설계(CPTED, WMS)](https://www.data.go.kr/data/15101832/openapi.do)
  - 적합성: 중상
  - 이유: 범죄예방 환경설계 적용 지역을 지도형으로 제공
  - 권장 사용: `범죄예방환경설계` 또는 `CPTED` 메뉴
  - 비고: 위험도 자체보다는 예방 인프라/환경 요인 레이어에 가까움

## 3. 보조 후보

- [ ] `치안사고통계`
  - API: [행정안전부_생활안전지도 치안사고통계(전체, WMS)](https://www.data.go.kr/data/15101821/openapi.do)
  - 적합성: 보조
  - 이유: 최근 3년 치안사고 발생 건수 기반 통계형 WMS
  - 권장 사용: 메인 예방 레이어보다는 분석/통계 보조 레이어

- [ ] `아동안전지킴이집`
  - API: [경찰청_아동안전지킴이집 정보 서비스](https://www.data.go.kr/data/3052084/openapi.do)
  - 적합성: 보조
  - 이유: 위치 기반 안전거점 메뉴로 사용 가능
  - 권장 사용: `사회적 약자` 도메인과 더 잘 맞지만, `치안/보호 거점` 보조 메뉴로도 가능

- [ ] `보안등`
  - 후보 데이터:
    - [전국보안등정보표준데이터](https://www.data.go.kr/data/15017320/standard.do?recommendDataYn=Y)
    - 지자체 보안등 OpenAPI 다수
  - 적합성: 보조
  - 이유: 야간 보행 안전과 범죄 예방 인프라로 의미가 큼
  - 권장 사용: `야간 안전시설` 또는 `보안등 지도`
  - 비고: 전국 단일 공통 API보다는 표준데이터/지자체 API 조합이 현실적
