# 3D 한국 통합 관제 시스템 개발 계획서 (Project Plan)

## 1. 프로젝트 개요
- **목표:** 대한민국 전역 및 주요 거점의 다양한 안전/보안 요소를 실시간으로 모니터링하는 3D 기반 웹 관제 애플리케이션 개발.
- **주요 테마:** 미래지향적이고 직관적인 사이버 버그/국방 관제 스타일의 UI (어두운 배경, 형광 포인트 컬러).

---

## 2. 주요 관제 도메인 (기능 명세)

> **[체크리스트 범례]** `[데이터 확보] [개발 완료]` 형식
> - `[x] [ ]` : 공공데이터포털, ITS, Open API 등을 통해 **실제 데이터 연동이 가능한 항목**
> - `[ ] [ ]` : 군사 보안, 개인정보 등으로 실제 데이터 확보가 어려워 **가상(Mock) 데이터 시뮬레이션이 필요한 항목**

### 2.1 ✈️ 항공 및 공역 관제 (Aviation & Airspace)
- [x] [ ] **실시간 비행 경로 시각화:** 항공기 및 헬기의 3D 궤적 추적 (OpenSky Network 등 API 활용 가능)
- [ ] [ ] **미확인 비행물체 탐지:** 비인가 드론 및 미식별 기체 접근 시 경보 알림 (군사 기밀, 시뮬레이션 필요)
- [x] [ ] **공항 내/외부 혼잡도 관제:** 주요 공항 여객기 이착륙 스케줄 및 공항 주차장 실시간 혼잡 상태 모니터링 (공항공사 공공데이터 API)
- [x] [ ] **항공기 주변 소음 등고선(Noise Contour):** 주요 공항 주변 3D 소음 맵핑 및 실시간 소음 측정망 연동 (공항공사 관측 API 활용)
- [x] [ ] **비행금지구역(No-Fly Zone) 표시:** 주요 시설 상공 무단 진입 시 경고망 활성화 (공공 공간정보 V-World 활용)

### 2.2 📷 실시간 영상 보안 관제 (CCTV Surveillance)
- [x] [ ] **CCTV 3D 맵핑:** 전국 주요 거점 CCTV 위치 3D 폴리곤 위에 마커 렌더링 (국가교통정보센터/지자체 API)
- [x] [ ] **실시간 스트리밍 연동:** 마커 클릭 시 대상 CCTV의 실시간 영상 팝업 제공 (ITS 국가교통정보센터 실시간 HLS)
- [ ] [ ] **이상 행동 감지 (AI 기반):** 화재, 폭력 등 위험 상황 발생 지점 점멸 효과 부여 (가상 연출 필요)
- [ ] [ ] **사각지대 분석:** 3D 건물 그림자를 활용한 카메라 시야각(FOV) 및 감시 사각지대 시각화 (프로그램적 계산/가상 구현)

### 2.3 🚗 도로 교통 및 물류 관제 (Highway Traffic)
- [x] [ ] **실시간 고속도로 병목 감지:** 도로 구간별 혼잡도 색상 및 애니메이션 적용 (국가교통정보센터 ITS 소통정보)
- [x] [ ] **도로전광표지(VMS) 실시간 송출 관제:** 전국 고속도로 VMS 위치별 현재 송출 중인 안내 문구/이미지 표출 (한국도로공사 API)
- [x] [ ] **고속도로 휴게소 혼잡도 및 충전소 상태:** 휴게소별 주차 혼잡도 및 수소/전기차 충전 인프라 가동 상태 맵핑 (한국도로공사 API)
- [x] [ ] **돌발 상황 및 도로 공사 표시:** 교통사고, 도로 공사, 도로 유실 위험 구역 3D 모니터링 (국가교통정보센터 돌발 API)
- [ ] [ ] **우회 경로 시뮬레이션:** 길막힘 발생 시 대체 우회 고속도로 추천 경로 시각화 (경로 계산 로직 필요)
- [x] [ ] **기상 악화 대응:** 폭설, 빙판길 우려 구간 경고 아이콘 노출 (기상청 공공데이터 API)

### 2.4 🛡️ 국방 및 영토 안보 관제 (Territorial Invasion / Security)
- [ ] [ ] **영해 및 영공 침범 알림:** 방공식별구역(KADIZ) 및 적대 세력 침투 시 붉은색 경계망 활성화 (군사 민감 정보, 가상 시나리오)
- [ ] [ ] **군사 분계선(MDL) 경계 상황:** 해안선 철책, 전방 부대 초소 감시망 및 과학화 경계 시스템 작동 상태 표시 (가상 데이터)
- [x] [ ] **미확인 지뢰 / 불발탄(UXO) 경고 구역:** 과거 전투 지역 등 특정 위험 지대에 대한 3D 경고 폴리곤 영역 렌더링 (공공 공간정보 API)
- [ ] [ ] **요격 자산/방어 시뮬레이션:** 침투 라인에 따른 자산 출격 요격 궤적 가시화 (가상 시뮬레이터 구성)

### 2.7 ⚓ 해양 및 항만 물류 관제 (Maritime & Port Logistics)
- [x] [ ] **해상교통관제(VTS) 모니터링:** 전국 무역항별 선박 관제 교신 현황 및 입출항 선박 실시간 감시 (해양수산부 관제정보 API)
- [x] [ ] **정박지 및 항만 터미널 혼잡도:** 주요 무역항(부산/인천 등) 터미널 컨테이너 혼잡 지표 및 주변 대기 선박 클러스터링 (해양수산부 API)
- [x] [ ] **실시간 해양 위험 구역 경고:** 해상 항만 공사 구역, 사격 훈련 구역, 암초 지대 접근 시 대상 객체에 Red Alert 표출 (해양수산부 해상정보 API)
- [x] [ ] **상선 및 어선 식별 (AIS 기반):** 해상 상선/어선 트래킹을 통한 개별 선박의 이동 속도, 톤수 등 상세 정보 패널 바인딩 (오픈형 AIS API)

### 2.5 🚨 재난/재해 및 환경 모니터링 (Disasters & Environment)
- [x] [ ] **산불 확산 시뮬레이션:** 산불 발생 위치 및 피해 범위 표시 (산림청 산불위치 API)
- [x] [ ] **수자원 관제:** 댐 수위 및 하천 범람 위험 구역 표시 (수자원공사 공공데이터 API)
- [x] [ ] **지진/대피소 안내:** 지진 발생 시 파동(Ripple) 효과 및 최단 거리 대피소 맵핑 (기상청 및 행안부 API)

### 2.6 🌤️ 기상 및 대기 환경 관제 (Weather & Atmosphere)
- [x] [ ] **전국 날씨 및 기상 이변 시각화:** 실시간 날씨 데이터(태풍 경로, 폭우 레이더 등) 레이어 중첩 표시 (기상청 단기예보/초단기실황 API)
- [x] [ ] **미세먼지/초미세먼지(AQI) 3D 대기 모델링:** 권역별 미세먼지 농도에 따른 3D 공간 연무(Fog) 및 파티클 색상(나쁨-붉은색) 적용 (환경공단 에어코리아 API)
- [x] [ ] **국지적 집중호우 경보:** 강수량 임계치 초과 지역에 대한 수직 기둥(Column) 형태의 3D 데이터 시각화 (기상청 API)
- [x] [ ] **바람장(풍향/풍속) 파티클 애니메이션:** 전국적인 바람의 흐름을 보여주는 동적 파티클 흐름선(Flow Map) 구현 (기상청 데이터 연동)

### 2.8 🚔 치안 및 범죄 예방 관제 (Crime & Public Safety)
- [x] [ ] **지역별 범죄 위험도 3D 히트맵:** 경찰청 지역별 범죄 발생 통계를 기반으로 우범 지대 및 범죄율 높은 구역을 3D 점밀도/히트맵으로 시각화 (경찰청 API)
- [x] [ ] **시간대/요일별 범죄 발생 확률 분석:** 시간의 흐름(UI 상의 가상 시계)에 따라 취약 시간대 범죄 발생 확률이 높은 지역을 경고색으로 강조 (경찰청 범죄 발생 시간대 통계 활용)
- [x] [ ] **성범죄자 거주 분포 클러스터링:** 성범죄자알림e 기반 공개/고지 지번 주소를 활용하여 고위험군 거주지 반경 내 학교/어린이집 등 취약 시설 접근성 경고 (여성가족부 관련 API)
- [ ] [ ] **실시간 112/119 출동 경로 시뮬레이션:** 긴급 상황 발생 시 경찰차/소방차의 예상 최적 출동 경로 및 도착 시간 3D 표시 (가상 데이터 적용)

### 2.9 🚋 대중교통 및 대규모 군중 밀집 관제 (Crowd Safety & Rail Transit)
- [x] [ ] **주요 지하철/KTX 실시간 트래킹:** 전국 철도망 및 주요 열차의 실시간 3D 위치 및 지연 상태 (철도 공공데이터 API)
- [ ] [ ] **군중 밀집도 및 압사 위험 경보:** 강남역, 홍대, 축제 지역 등 유동 인구 밀집 구역의 3D 히트맵 및 위험 단계(주의/경고/위험) 점멸 알림 (통신사 기지국 데이터 기반 시뮬레이션)

### 2.10 🏥 보건/의료 및 응급 관리 체계 (Healthcare & Emergency)
- [x] [ ] **실시간 구급차 출동 궤적:** 112/119 출동 경로와 더불어, 환자 이송 중인 구급차 위치 표시
- [x] [ ] **응급실/중환자실 병상 현황 (NEDIS 연동):** 전국 주요 거점 병원의 응급실 수용 가능 여부 패널 (국립중앙의료원 API 등)
- [ ] [ ] **감염병/전염병 확산 시각화:** 조류독감(AI), 구제역, 또는 신종 감염병 발생지 반경 및 방역 통제선 표시 (가상 데이터 시뮬레이션)

### 2.11 ⚡ 국가 기반 인프라 관제 (Critical Infrastructure)
- [x] [ ] **에너지/전력망 상태:** 주요 발전소/변전소별 전력 부하량 점진적 표시 및 블랙아웃(정전) 발생 지역 쉐이딩 처리 (한국전력 API)
- [x] [ ] **방사능 수치 및 원전 감시:** 전국 생활 방사능 수치 및 주요 원자력 발전소 주변 상태 모니터링 (원자력안전위원회 API)
- [x] [ ] **수자원/식수 오염:** 주요 정수장 및 상하수도 이상 발생 경보 (수자원공사 API 등)

### 2.12 💻 사이버 안보 관제 (Cybersecurity & Network)
- [ ] [ ] **실시간 DDoS 및 해킹 공격 궤적:** 해외(중국, 러시아 등)에서 한국 주요 관공서 서버로 쏟아지는 사이버 공격망을 3D 곡선(Arc) 빔으로 표현 (시뮬레이션)
- [ ] [ ] **주요 통신망 장애 구역:** SKT/KT/LGU+ 통신 마비 지역 렌더링 (가상 시나리오)

### 2.13 🧑‍🦯 사회적 약자 보호망 (Vulnerable Populations)
- [ ] [ ] **실종 아동 / 치매 노인 경계망 (Amber Alert):** 발생 위치 기준 예상 이동 반경을 3D 파동(Ripple)으로 지속 표출 (가상 데이터/경찰청 연동)
- [ ] [ ] **독거노인/중증 장애인 응급 알람:** 댁내 IoT 센서 무반응 시 지도상 위급 신호 생성 (가상 데이터)

## 3. 단계별 개발 로드맵

### Phase 1: 3D 기본 환경 및 관제 UI 인프라 구축 (현재 단계)
- [ ] Next.js 기반 프로젝트 라우팅 및 기본 레이아웃 구성.
- [ ] 3D 지도 라이브러리(Mapbox GL JS, Deck.gl 또는 CesiumJS) 설정 및 한국 중심으로 지도 세팅.
- [ ] 다크 모드 기반의 프리미엄 관제 대시보드 UI/UX 구축 (TailwindCSS, Framer Motion 활용).

### Phase 2: 정적 데이터(Point of Interest) 시각화 
- [ ] 전국 주요 CCTV 시스템 마커 도포 및 지도 확대/축소에 따른 클러스터링 구현.
- [ ] 국가 주요 시설 위험도 상태 표시.
- [ ] 카메라 시점 이동 액션 (선택 객체로 부드럽게 Zoom-In).

### Phase 3: 동적 데이터 연동 및 애니메이션 (Real-time Simulation)
- [ ] 가상의 항공기/드론 3D 모델 및 이동 궤적 애니메이션 추가.
- [ ] 고속도로 교통량 변화에 따른 실시간 선(Line) 애니메이션 반영.
- [ ] 우측/좌측 상태 패널에 선택된 객체의 실시간 데이터(속도, 위/경도, 영상 등) 바인딩.

### Phase 4: 이벤트 기반 경보 시스템 및 최적화
- [ ] **영토 침범 / 대규모 추돌 사고 등 위협 스크립트 실행 시각화.**
- [ ] 화면 전체를 붉게 감싸는 Warning Overlay UI 연동.
- [ ] 대량의 마커/객체 렌더링에 대한 WebGL 성능 최적화.
